Lab 1: Intro to Logic Simulation

-----------
DESCRIPTION

In this lab, the user flips input switches to match the logic of truth tables in 3 different parts.
The truth tables are given in the Lab 1 explanation document.


-----------
FILES

-
Lab1.lgi

This file includes the simulation of circuits that ouputs the logic of the truth tables.

-----------
INSTRUCTIONS

This program is intended to be run using MultiMedia Logic(MML). Press the green "play" button to
run the program, and flip the switches to simulate the logic. 0 means the switch is off and 1 means the switch is on. 
Press the red square button to stop the simulation.
