Lab 3: ASCII-risks (Asterisks)

-----------
DESCRIPTION

In this lab, the user enters an integer value greater than 0. The program will print 
a triangle of asterisks starting at the second row.

-----------
FILES

-
Lab3.asm

This file includes the assembly code of the lab.

-----------
INSTRUCTIONS

This program is intended to be run using the MIPS Assembler and Runtime Simulator
(MARS). Enter the test case as a program argument and run using MARS.
