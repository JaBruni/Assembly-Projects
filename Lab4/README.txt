Lab 4: Functions and Graphs

-----------
DESCRIPTION

This program allows users to change the background of the bitmap display, 
and "draw" horizontal and vertical lines.

-----------
FILES

-
lab4.asm

This file includes the assembly code of the lab.

-----------
INSTRUCTIONS

This program is intended to be run using the MIPS Assembler and Runtime Simulator
(MARS). Enter the test case as a program argument and run using MARS. Use the
bitmap display tool to see the color output. Set the display height and width to
128 x 128 and the base address to 0xffff0000.
